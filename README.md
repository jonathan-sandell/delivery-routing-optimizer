# Delivery Routing Optimizer

## Overview
A Python application that simulates package delivery logistics using route optimization and time-based package tracking.

## Features
- Route optimization
- Package status tracking
- Mileage calculation
- Multi-truck delivery simulation
- Custom hash table implementation

## Technologies
- Python
- Object-Oriented Programming
- CSV Processing

## Run Locally

```bash
git clone https://github.com/jonathan-sandell/delivery-routing-optimizer.git
cd delivery-routing-optimizer
python main.py
```
